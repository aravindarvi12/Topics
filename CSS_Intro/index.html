A paragraph is a collection of words combined together to make a longer unit than a sentence. It's a set of sentences that are well-organized and coherent, and they're all about the same thing. Several sentences often make up a paragraph. There are normally three to eight sentences in a paragraph. Almost all of the writing one reads in a textbook or an article can be grouped into paragraphs if it is longer than a few sentence.
